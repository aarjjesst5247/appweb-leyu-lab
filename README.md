# appweb-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能涉及不同的主题、工具或演示内容，彼此之间没有强关联。

## 项目简介

这是一个静态 HTML 页面集合仓库。每个页面均为独立的归档文件，可以直接通过浏览器打开查看。仓库本身不绑定特定域名或网站，仅作为页面存储与分发的统一位置。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面入口（如有）
├── pages/              # 存放主要页面文件
├── assets/             # 公共资源（图片、样式、脚本等）
├── docs/               # 额外文档或说明
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名通常反映其内容或用途。
- `assets/`：页面引用的资源文件，按需组织。
- `docs/`：可选的补充文档。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可离线或在线直接运行。
- 页面之间不共享状态或数据，除非明确说明。
- 部分页面可能依赖外部资源（如 CDN 库），请确保网络可达。
- 页面内容可能随时间更新或替换，历史版本可查看 Git 提交记录。
